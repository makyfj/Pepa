### Pepa

Pepa is a workout app that allows to keep track of your workout journey :)!

## How to run it

1. Clone the project: `git clone https://github.com/makyfj/Pepa`
2. Install dependencies: `yarn` or `npm i`
3. Run the app: `yarn start`
